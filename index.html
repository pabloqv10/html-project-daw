Herramientas para el desarrollo de apliaciones WEB

A la hora de desarrollar una aplicacion web, contamos con multiples herramientas que nos pueden hacer el desarrollo menos complejo, por ejemplo:

  1. React
  2. Vue
  3. Angular
  4. Svelte

React

¿Qué es React?

React es una biblioteca de JavaScript declarativa, eficiente y flexible para construir interfaces de usuario. Permite componer IUs complejas de pequeñas y aisladas piezas de código llamadas “componentes”.

¿Cuáles son sus principales ventajas?

  • DOM virtual
  • Isomorfismo
  • Componentes
  • Flujo de datos unidireccional
  • Amplia comunidad

Vue

¿Qué es Vue?

Vue JS es un framework de JavaScript para el desarrollo de interfaces de usuario y aplicaciones de una sola página. Fue creado por Evan You, un ex trabajador de Google, en 2014. El objetivo principal de Vue JS es simplificar el desarrollo web de las app y los sitios web. Gracias a su pequeño tamaño (apenas 18 kb), su facilidad de uso y aprendizaje, y rendimiento, se ha convertido en uno de los framework de JavaScript más populares.

¿Cuáles son sus principales ventajas?

  • Fácil de aprender
  • Es rápido
  • Es ligero
  • Es flexible
  • Flujo de datos bidireccional

Angular

¿Qué es Angular?

Angular es un framework de ingeniería de software de código abierto mantenido por Google, que sirve para desarrollar aplicaciones web de estilo Single Page Application (SPA) y Progressive Web App (PWA). Sirve tanto para versiones móviles como de escritorio.

¿Cuáles son sus principales ventajas?

  • Uso de DOM regular
  • Enlace de datos o data binding
  • Compatibilidad móvil y de escritorio
  • Velocidad y rendimiento
  • Productividad
  • Directivas

Svelte

¿Qué es Svelte?

Svelte (o SvelteJS) es un framework para desarrollar aplicaciones JavaScript. Proporciona el marco básico para la aplicación a desarrollar, así como funciones importantes y recurrentes en forma de bibliotecas integradas automáticamente. El framework define las posibilidades básicas de la aplicación estableciendo un marco concreto y también influye en el proceso de desarrollo de manera decisiva. Familiarizarse con los frameworks diseñados para programar aplicaciones web no lleva mucho tiempo, por lo que estos entornos son fáciles de usar.

¿Cuáles son sus principales ventajas?

  • Sencillo desde la perspectiva del programador
  • Compilación rápida de componentes
  • Consola ágil
  • Sintaxis simple
  • CSS por componentes o en modo global
  • Sin Typescript
